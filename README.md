# Titanic Survival Prediction

![Titanic](https://img.shields.io/badge/Project-Titanic%20Survival-blue) ![Machine Learning](https://img.shields.io/badge/Machine-Learning-orange) ![Python](https://img.shields.io/badge/Python-3.8%2B-green)

Проект по предсказанию выживаемости пассажиров Титаника в рамках соревнования на **Kaggle**.

## Описание проекта

Этот проект решает задачу бинарной классификации - предсказание, выжил ли пассажир Титаника (1) или нет (0). Используются различные методы предобработки данных, feature engineering и алгоритмы машинного обучения.

## Метрика оценки

**Accuracy** - процент правильно предсказанных исходов:

```
Accuracy = (Количество правильных предсказаний) / (Общее количество пассажиров)
```
## Быстрый старт

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск пайплайна

```bash
# Предобработка данных
python src/data_preprocessing.py

# Обучение модели
python src/model.py

# Предсказание на тестовых данных
python src/predict.py
```

## Основные этапы

### 1. Предобработка данных
- Обработка пропущенных значений
- Кодирование категориальных переменных
- Масштабирование числовых признаков

### 2. Feature Engineering
- Извлечение признаков из имени (Title)
- Создание семейных признаков (FamilySize)
- Группировка возрастов и тарифов

### 3. Моделирование
- Логистическая регрессия
- Random Forest
- Gradient Boosting (XGBoost, CatBoost)
- SVM

### 4. Валидация
- Кросс-валидация
- Подбор гиперпараметров
- Анализ важности признаков

## Результаты

Лучшая модель достигает accuracy **0.78-0.82** на публичном лидерборде.
